# ROS 2 Topics

This document defines the key ROS 2 topics that will be used in the textbook examples and the capstone project.

| Topic Name | Message Type | Description | Publisher(s) | Subscriber(s) |
|---|---|---|---|---|
| `/voice_command` | `std_msgs/msg/String` | Carries the transcribed text from a voice command. | `voice_recognition_node` | `llm_planner_node` |
| `/joint_states` | `sensor_msgs/msg/JointState` | Publishes the current state (position, velocity, effort) of all robot joints. | `robot_state_publisher` or Simulator Bridge | `control_node` |
| `/cmd_vel` | `geometry_msgs/msg/Twist` | Sends velocity commands to the mobile base of the robot. | `navigation_stack` | `robot_base_driver` |
| `/scan` | `sensor_msgs/msg/LaserScan` | Provides LiDAR data for navigation and obstacle avoidance. | Simulator Bridge or LiDAR Driver | `navigation_stack` |
| `/odom` | `nav_msgs/msg/Odometry` | Publishes the estimated odometry of the robot. | `robot_base_driver` | `navigation_stack` |
| `/tf` | `tf2_msgs/msg/TFMessage` | Publishes the transformations between coordinate frames. | `robot_state_publisher`, `slam_toolbox` | Various |
| `/tf_static` | `tf2_msgs/msg/TFMessage` | Publishes static transformations that do not change over time. | `robot_state_publisher` | Various |
| `/depth_camera/image_raw` | `sensor_msgs/msg/Image` | Raw image stream from the depth camera. | Simulator Bridge or Camera Driver | `perception_node` |
| `/depth_camera/points` | `sensor_msgs/msg/PointCloud2` | Point cloud data generated from the depth camera. | `depth_image_to_pointcloud_node` | `perception_node`, `slam_toolbox` |
| `/llm_plan` | `std_msgs/msg/String` | Contains the high-level plan generated by the LLM. | `llm_planner_node` | `task_executor_node` |

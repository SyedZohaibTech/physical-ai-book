# Physical AI & Humanoid Robotics Book with RAG Chatbot Constitution

## Core Principles

### Accuracy
Reference official docs and peer-reviewed sources.

### Clarity
Understandable for CS/robotics learners.

### Reproducibility
All code and simulations must run.

### Rigor
Use authoritative sources.

## Additional Constraints

### Standards
- Traceable claims (APA style).
- ≥50% peer-reviewed sources.
- 0% plagiarism.
- Flesch-Kincaid grade 10-12.
- Code must compile/run.

### Constraints
- Word count: 7,000–10,000.
- Min 20 sources.
- Docusaurus book on GitHub Pages.
- RAG chatbot with FastAPI, Neon Postgres, Qdrant.

## Development Workflow

### Success Criteria
- Complete all modules with examples.
- Code verified.
- Chatbot answers accurately.
- Zero plagiarism.
- Deployed and live.

## Governance

This constitution outlines the fundamental principles and guidelines governing the "Physical AI & Humanoid Robotics Book with RAG Chatbot" project. It supersedes all other conflicting practices or documents. Amendments to this constitution require a formal documentation process, approval from relevant stakeholders, and a clear migration plan for any affected components. All pull requests and code reviews must verify compliance with these principles.
